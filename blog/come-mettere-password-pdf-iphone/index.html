Come mettere una password su un PDF da iPhone

Di Giulio Palumbo Schiavone · Pubblicato il 18 aprile 2026 · 7 min di lettura

In breve — iOS non ha un pulsante nativo per “aggiungere password a un PDF”. Hai tre strade realistiche: l'esportazione da Pages, un comando rapido via Shortcuts, o un'app dedicata. Per la maggior parte dei casi, solo l'app dedicata funziona con qualsiasi PDF in pochi secondi.

Stai per mandare per email un contratto, una busta paga o un referto medico in formato PDF. Vuoi che sia protetto con una password: anche se l'email viene inoltrata o finisce nel posto sbagliato, il contenuto deve restare privato. Su iPhone, sorprendentemente, non è un'operazione da un tocco. Ecco cosa funziona davvero nel 2026.

Perché proteggere un PDF con password?

Un PDF senza password è in sostanza un documento in chiaro: chiunque lo riceva — volutamente o per errore — può aprirlo, leggerlo, copiarlo e inoltrarlo. Per un menù di un ristorante va benissimo. Non va bene invece per:

  • Contratti, NDA, offerte commerciali
  • Documenti fiscali, buste paga, estratti conto
  • Scansioni di carta d'identità, passaporto, permessi di soggiorno
  • Referti medici e prescrizioni
  • Tutto ciò che non vorresti leggere su un cartellone pubblicitario

Aggiungere una password crittografa il file. Senza la password il contenuto è illeggibile — non semplicemente “nascosto dietro un login”, proprio cifrato. È la protezione che serve quando il file può uscire dal tuo controllo.

Opzione 1 — Usare Pages (gratis, macchinoso)

L'app Pages di Apple può esportare un documento come PDF protetto da password. È gratis, ma funziona solo se parti da un documento Pages, non da un PDF esistente. Quindi:

  1. Apri o importa il PDF in Pages (verrà convertito: la formattazione può saltare).
  2. Tocca … → Esporta → PDF.
  3. Attiva Richiedi password e imposta la password.
  4. Salva o condividi il file risultante.
Attenzione: Pages tratta il PDF come un documento modificabile. I layout complessi (moduli, fatture multi-colonna, documenti scansionati) possono uscire con testo spostato o elementi mancanti. Confronta sempre prima e dopo.

Opzione 2 — Un comando rapido di Shortcuts (gratis, limitato)

L'app Comandi Rapidi di iOS ha un'azione Cifra PDF. Puoi costruire un comando da foglio di condivisione che chiede una password e salva la copia cifrata. Pro: gratis, funziona su qualsiasi PDF. Contro: devi costruirlo e mantenerlo tu, non offre controlli su password di proprietario, permessi o algoritmo (usa quello di default del sistema).

Se ti serve una volta ogni morte di papa, è una strada ragionevole. Se blocchi PDF regolarmente, la frizione di ogni utilizzo diventa presto fastidiosa.

Opzione 3 — Un'app dedicata (rapida, funziona con tutti i PDF)

Se vuoi il flusso “scelgo un PDF, metto una password, fatto”, serve un'app dedicata. Qui sotto vediamo come funziona con Unlock my PDF, ma qualsiasi app seria sull'App Store dovrebbe fare sostanzialmente lo stesso.

  1. Apri l'app e scegli lo strumento Blocca.
  2. Seleziona il PDF da File, iCloud Drive, Google Drive, Dropbox o dal foglio di condivisione di un'altra app.
  3. Imposta una password che ricorderai. Una buona password: almeno 10 caratteri, maiuscole e minuscole, un paio di cifre, un simbolo. Evita riferimenti pubblici (nome, data di nascita).
  4. Salva la copia protetta. L'app genera un nuovo PDF cifrato. L'originale resta intatto a meno che tu non lo sovrascriva.
  5. Condividi il file cifrato. Inviaglielo via email, AirDrop, upload. Chi lo apre dovrà inserire la password.
Dove finisce la password? In un'app seria, tutto avviene sul dispositivo. Il file non lascia mai l'iPhone, la password non viene mai caricata online. Prima di usare una qualsiasi app PDF, controlla la privacy policy: se un servizio online gratuito ti chiede di caricare il PDF sul suo server, è una bandiera rossa per documenti sensibili.

Come scegliere una password che non rimpiangerai

La password di un PDF non è recuperabile. Se la perdi, il file è perso — non c'è un link “password dimenticata”. Due regole pratiche:

  • Non inventare password al volo. Usa il tuo password manager (Portachiavi iCloud, 1Password, Bitwarden) per generarla e memorizzarla.
  • Condividi la password su un canale diverso. Se mandi il PDF per email, invia la password via iMessage, Signal o a voce. Mai nella stessa email.

Domande frequenti

Posso proteggere con password un PDF direttamente dall'app File?

No. Al momento, l'app File di iOS non offre un'opzione integrata per aggiungere password a un PDF. Serve Pages, un comando rapido, o un'app dedicata.

Una password su un PDF è davvero sicura?

Dipende dalla crittografia e dalla password stessa. Le app PDF moderne usano AES-256 di default, lo stesso standard delle immagini disco cifrate e della messaggistica sicura. Con una password forte, il file è effettivamente illeggibile senza di essa. Approfondimento: PDF encryption explained — 128-bit vs 256-bit (in inglese).

Posso rimuovere la password dopo, se me la ricordo?

Sì. Se conosci la password puoi aprire il file ed esportarlo senza protezione. Vedi How to remove a known password from a PDF on iOS.

E se il destinatario usa Android o Windows?

Un PDF protetto con password standard si apre su qualsiasi piattaforma. Adobe Reader, Anteprima su macOS e la maggior parte dei browser chiederanno la password. Non serve software aggiuntivo.

In conclusione. iOS offre delle scorciatoie, non una soluzione nativa vera. Se gestisci PDF sensibili più di un paio di volte all'anno, il minuto speso a installare un'app dedicata si ripaga la prima volta che mandi qualcosa di privato senza doverci pensare due volte.
